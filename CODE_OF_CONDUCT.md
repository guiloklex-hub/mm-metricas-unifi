# Código de Conduta

Este projeto adota o [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Resumo prático

- Seja respeitoso. Discorde de ideias, não de pessoas.
- Aceite crítica construtiva. Ofereça crítica construtiva.
- Não tolere assédio, discriminação ou comportamento intimidador.
- Privacidade: nada de dados pessoais alheios sem permissão.

## Aplicação

Comportamento inaceitável pode ser reportado a `conduct@<dominio-do-mantenedor>` (substituir quando definido). Mantenedores investigam e podem aplicar:

1. Aviso privado
2. Aviso público
3. Suspensão temporária de interação
4. Banimento permanente

## Escopo

Issues, PRs, discussões, canais de chat e qualquer espaço onde o projeto é discutido.
